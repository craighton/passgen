# PassGen - Password Generator

A client-side password generator with no dependencies and no server requests. All generation happens in the browser.

## Features

- Configurable password length (6–50 characters)
- Character set options: lowercase, uppercase, numbers, symbols
- Customizable symbol set
- Constraints: no similar characters, no duplicates, no sequential characters, begin with a letter
- Generate multiple passwords at once (up to 50)
- Settings saved via cookies
- Auto-generate on page load

## Usage

Open `index.html` in any modern browser. No build step or server required.
